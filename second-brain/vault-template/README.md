# この vault について（人間向け）

これは **Obsidian で開くセカンドブレイン**です。Obsidian はこのフォルダの上に乗る
無料アプリで、`[[wikilinks]]` とグラフビューだけ使えれば十分。プラグインもコネクタも
特別な設定も不要です。**あなたはアプリを、エージェントはフォルダを**、同じ脳として見ます。

- 中身の書き方・読み方のルールは `CLAUDE.md`（エージェントが毎回読む）を参照。
- 何があるかは `INDEX.md`（玄関）を参照。
- 素材は `raw/` に無加工で入れる（読み取り専用）。まとめは `entities/` と `concepts/`。

## 保存（同期）は git に一本化する

**vault が死ぬのは同期の競合が原因**です。iCloud/Dropbox とエージェントの書き込みが
かち合うと、conflicted copy とフォルダ崩れが起きます。保存点は **git** に一本化して
ください（`new-vault.sh` が初期化済み）。節目でコミット:

```bash
cd <この vault> && git add -A && git commit -m "..."
```

キットのループ（セッション採掘 / 夜間コンパイル / 週次 lint・総合）は、書き込み後に
自動で git チェックポイントを打ちます。
