# raw/ — 素材（読み取り専用・地の真実）

ここに入れたものは**無加工の履歴**です。エージェントは raw/ を**読むだけ**で、
書き換えません。細部がぼやけて誤差が積み上がるのを防ぐため、素材と compiled
（entities/concepts）は常に分けます。

## 何を入れるか

- 記事・ブログ・ドキュメントの本文
- 通話 / 会議の文字起こし
- ブックマークしたスレッド、競合ページ
- メモアプリのエクスポート、クライアントフォルダ、過去リサーチ

## 自動で増えるサブフォルダ

- `sessions/` … `session-mine.sh`（SessionEnd フック）が採掘したセッション記録
- `research/` … `research.sh`（リサーチ機）が着地させた検証済み・日付き・出典付きページ
- `maintenance/` … `lint.sh`（週次 lint）の点検レポート

これらは日付きで着地し、夜間の `compile.sh` が entities/concepts へ昇格させます。
INDEX.md には compiled ページだけを載せ、raw/ は各ページの出典リンクから辿ります。
