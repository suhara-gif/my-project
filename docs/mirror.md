  唯一、実データ(実カレンダー、実広告費、実KPI障害)を伴って運用され続けている。

一番作り込んだものが一番使われず、一番地味なものが唯一生きている。
`fable5-agent-system` の README 冒頭は、着想元にした X の記事の事実誤認を訂正する文章から
始まっている — 実務の必要からではなく、記事に触発されて作った可能性が高い。

**ルールは、壊れた後にしか書かれていない。**
claude-backup は 2026-06-06〜07 に一気に作られ、直後2コミットが
「Fix macOS portability」「Fix unbound-variable crash」。そこから34日間このリポジトリに

「壊れて学ぶ」を止めるのではなく学びを回収する仕組みの方を強化する選択をしている。
